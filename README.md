# weekly_report

週報を記録・コピーできる Flutter Web アプリ。

## URL

https://vazaluta.github.io/weekly_report/

## 機能

- 週報の10問フォーム（振り返り・目的の見直し・来週の目標設定）
- 入力内容をMarkdown形式でクリップボードにコピー
- テキスト選択コピー対応

## 技術スタック

- Flutter 3.32.2
- GitHub Actions（自動ビルド・デプロイ）
- GitHub Pages（ホスティング）

## デプロイ

`main` ブランチに push すると GitHub Actions が自動で Flutter Web をビルドし、GitHub Pages にデプロイされます。
